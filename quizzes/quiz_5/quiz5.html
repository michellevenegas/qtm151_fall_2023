(c) Compute growth rates

  • Calculate a new column "growth_euro" which computes the percentage daily
    growth rate of "eur_to_usd".

Hint: Use .diff() and .shift(1) as in Lecture 21

In[]:
# Write your own code
currencies["eur_diff"] = currencies["eur_to_usd"].diff()
currencies["eur_lag"] = currencies["eur_to_usd"].shift(1)
currencies["growth_euro"] = currencies["eur_diff"]*100/currencies["eur_lag"]

(e) Aggregate dataset by time period

  • Follow the instructions in Lecture 20 to compute an aggregate dataset
    by month which computes the standard deviation of "eur_to_usd"
  • Display the dataset
In[]:
# Write your own code
month_config = pd.Grouper(key ="date", freq = "m")
monthly_data = (currencies
                .groupby(month_config)
                .agg(eur_std = ("eur_to_usd", "std")))
